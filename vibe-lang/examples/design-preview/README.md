# Design Preview Examples

These examples showcase planned v1+ features that are **not yet implemented**:
- `@entity`, `@scene`, `@on`, `@component` annotations
- `spawn()`, `destroy()`, `tween()`, `camera_shake()`
- `yield`, coroutines, scene transitions
- `play_sound()`, `play_music()`

They demonstrate the vision for Vibe's game engine layer. Do not attempt to run these — they will not parse or compile.

For working examples, see `demo_bouncing_balls.vibe` in the parent directory.
